# Versión Portable (Ejecutable Windows)

Esta rama contiene la versión compilada y **totalmente portable** del Cifrador Híbrido. 
Incluye su propio entorno Java (JRE) y las librerías necesarias, por lo que **no requiere instalación previa** de ningún software.

## Instrucciones de Instalación

Una vez descargado el archivo `.zip` del enlace anterior:

1.  **Descomprimir:** * Haz clic derecho sobre el archivo descargado.
    * Selecciona **"Extraer todo..."** o "Descomprimir aquí".
    * **Importante:** No intentes ejecutar el programa sin descomprimirlo primero; fallará porque no encontrará las carpetas `bin` y `libs`.

2.  **Ubicar la carpeta:**
    * Entra en la carpeta extraída (ej. `CifradorHibrido/`).

3.  **Ejecutar:**
    * Haz doble clic en el archivo **`Cifrador.exe`** (la del ícono del candado amarillo).
